Front-End Web Development with React

Este curso forma parte de Programa especializado: Full-Stack Web Development with React
ofrecido por Universidad Científica y Tecnológica de Hong Kong

sobre este curso.

Este curso explora el desarrollo de aplicaciones front-end basadas en Javascript y, en particular, la biblioteca React.
Este curso utilizará JavaScript ES6 para desarrollar la aplicación React. También recibirá una introducción al uso de React strap para el diseño de interfaz de usuario receptivo basado en Bootstrap 4. Se le presentarán varios aspectos de los componentes de React. Aprenderá sobre el enrutador React y su uso en el desarrollo de aplicaciones de una sola página. También aprenderá a diseñar formularios controlados. Se le presentará la arquitectura Flux y Redux. Explorará varios aspectos de Redux y lo usará para desarrollar aplicaciones impulsadas por React-Redux.
Al final de este curso:

    - Familiarícese con el desarrollo de aplicaciones Javascript del lado del cliente y la biblioteca React
    - Ser capaz de implementar aplicaciones de una sola página en React
    - Poder utilizar varias funciones de React, incluidos componentes y formularios
    - Ser capaz de implementar una aplicación web front-end funcional usando React
    - Ser capaz de usar Reactstrap para diseñar aplicaciones React receptivas
    - Poder usar Redux para diseñar la arquitectura de una aplicación React-Redux.
