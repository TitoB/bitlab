Acerca de este Curso
326.044 vistas recientes

‎Este curso le dará una visión general de los marcos de interfaz de usuario web del lado del cliente, en particular Bootstrap 4. Aprenderá sobre cuadrículas y diseño receptivo, Bootstrap CSS y componentes JavaScript. Aprenderá sobre los preprocesadores CSS, Less y Sass. También aprenderá los conceptos básicos de Node.js y NPM y corredores de tareas como Grunt y Gulp.‎‎Al final de este curso, podrá:

 a) Configurar, diseñar y diseñar una página web utilizando Bootstrap 4 y sus componentes
 b) Crear un diseño de página web receptivo y 

c) Hacer uso de herramientas web para configurar y administrar sitios web. Este curso también incluye una pista de honores que le permite trabajar en su propio proyecto desarrollando un sitio web utilizando Bootstrap 4. Los estudiantes que se inscriban en este curso deben tener un buen conocimiento previo de HTML, CSS y JavaScript.‎

Bootstrap CSS Components

This module concentrates on Bootstrap's components that are designed using pure CSS classes. You will learn about Navigation and Navigation bar. Then, you will learn about buttons, forms, tables, cards, images and media, tags, alerts and progress bars. At the end of this module you need to complete your second assignment.

semana 3

Bootstrap Javascript Components



This module concentrates on Bootstrap's JavaScript based components. You will learn about tabs, pills and tabbed navigation, collapse, accordion, scrollspy, affix, tooltips, popovers, modals and the carousel. At the end of this module you need to complete the third assignment.

5 horas

Web Tools
This module rounds out our discussions on Bootstrap JavaScript components. Then we discuss CSS preprocessors, Less and Sass. Finally, we discuss building and deployment of our Web projects through task automation using NPM scripts, and task runners like Grunt and Gulp.
