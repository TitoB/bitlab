Desarrollo del lado del servidor  con Nodejs,Express y Mongo DB

Acerca de este curso

Este curso trata sobre todo lo relacionado con el servidor. Basamos todo el curso en torno a la plataforma NodeJS. Comenzamos con una breve descripción de los protocolos web: HTTP y HTTPS. Examinamos los módulos NodeJS y NodeJS: Express para construir servidores web. En el lado de la base de datos, revisamos las operaciones CRUD básicas, las bases de datos NoSQL, en particular MongoDB y Mongoose para acceder a MongoDB desde NodeJS. Examinamos los conceptos REST y la construcción de una API RESTful. Tocamos la autenticación y la seguridad. Finalmente, revisamos los enfoques de backend como servicio (BaaS), incluidos los BaaS móviles, tanto los servicios de código abierto como los BaaS comerciales.Al final de este curso, podrá: - Demostrar comprensión de los conceptos del lado del servidor, CRUD y REST - Construya y configure un servidor backend usando el marco NodeJS - Cree una API RESTful para que el front-end acceda a los servicios de backend
